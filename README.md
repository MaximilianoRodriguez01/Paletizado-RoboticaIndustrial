# Paletizado con Robot Industrial (ABB IRB140)

Simulación de un proceso de **paletizado** utilizando un **robot ABB IRB140**, desarrollado para visualización y análisis de automatización en robótica industrial. Esta misma representa una secuencia de recolección y colocación de tablas de madera sobre un pallet, con el objetivo de aprender acerca de automatización y control robótico.
